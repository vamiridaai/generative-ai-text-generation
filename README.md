
# Generative AI Text Generation using GPT Concepts

This repository contains an academic project that explores the fundamentals of **Generative Artificial Intelligence (AI)**, with a focus on **Generative Pre-trained Transformers (GPTs)** and practical text generation. The project combines theoretical understanding with hands-on implementation using Python and TensorFlow/Keras.

The work was developed and executed in **Google Colab** for reproducibility and ease of evaluation.

---

##  Project Objectives

- Understand the core concepts of Generative AI
- Study the architecture and functionality of GPT models
- Train a basic text generation model using a public dataset
- Demonstrate a practical content creation application
- Discuss ethical considerations related to Generative AI

---

##  Key Concepts Covered

- Generative vs Discriminative Models  
- Transformer Architecture  
- Self-Attention Mechanism  
- Tokenization and Sequence Modeling  
- Autoregressive Text Generation  
- Ethical Considerations in AI  

---

##  Dataset

**Source:** Project Gutenberg (Public Domain)  
**Example Text:** *Pride and Prejudice* by Jane Austen  

The dataset is automatically downloaded within the notebook to ensure reproducibility and to avoid file path issues.
